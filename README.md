## 📦 Install

1. Go to **Releases** tab
2. Download `Cancer.zip`
3. Disable Defender / add exclusion
4. Extract, launch CS2, run `Cancer.exe` as Admin
5. `INSERT` 

---

## ⌨️ Keys

| Key | Action |
|-----|--------|
| `INSERT` | Menu |
| `F6` | ESP toggle |
| `F7` | Aimbot toggle |
| `F8` | Triggerbot toggle |
| `F9` | Panic unload |
| `END` | Force exit |

## 🔧 Config
Stored in `%appdata%\\Cancer\\config.ini`
Scripts in `%appdata%\\Cancer\\scripts\\`
